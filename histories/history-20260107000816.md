# History - Task List Sorting by Order

## 날짜: 2026-01-07

## 작업 내용:
- `src/components/TaskList.tsx`에서 할 일 목록을 `order` 속성 기준 오름차순으로 정렬하여 표시하도록 수정.
- `Task` 인터페이스에 `order` 속성이 있음을 확인하였고, `App.tsx`에서 새로운 할 일 추가 시 `order`가 올바르게 할당됨을 확인.
- `TaskEditorModal.tsx`에서 할 일 수정 시 `order` 속성이 유지됨을 확인.

## 변경 사항:
- `src/components/TaskList.tsx`:
    - `tasks` 배열을 렌더링하기 전에 `[...tasks].sort((a, b) => (a.order ?? 0) - (b.order ?? 0))`를 사용하여 정렬된 배열 `sortedTasks` 생성.
    - 정렬된 `sortedTasks`를 사용하여 `TaskItem` 컴포넌트들을 매핑.
