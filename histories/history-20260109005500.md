# 작업 히스토리 - 2026-01-09 (8)

## 작업 내용
프로젝트 추가 및 수정 모달의 사용자 경험(UX)을 개선하기 위해 닫기 동작과 포커스 설정을 추가했습니다.

### 주요 변경 사항
1.  **모달 닫기 편의성 개선**
    *   **대상:** `AddProjectModal`, `EditProjectModal`
    *   **배경 클릭 닫기:** 모달 외부 영역(Dimmed background) 클릭 시 모달이 닫히도록 이벤트 핸들러를 추가했습니다.
    *   **ESC 키 닫기:** 키보드의 `Escape` 키를 누르면 모달이 닫히도록 `keydown` 이벤트 리스너를 등록했습니다.
2.  **프로젝트 수정 모달 포커스 개선** (`EditProjectModal`)
    *   모달이 열릴 때 프로젝트 이름 입력창에 자동으로 포커스가 가도록 `autoFocus` 속성을 추가했습니다. (기존 추가 모달에는 이미 적용됨)
