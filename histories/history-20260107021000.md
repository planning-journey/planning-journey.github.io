# 작업 내역 (2026-01-07)

## 요청 사항
- GitHub Pages에서 SPA 동작을 위해 `index.html`로 리다이렉트하는 `404.html` 생성.

## 수행 내용
1. **404 페이지 생성 (`public/404.html`)**:
   - `rafgraph/spa-github-pages` 방식을 사용하여 현재 경로를 쿼리 스트링으로 변환해 `index.html`로 보내는 스크립트를 작성했습니다.
   - 프로젝트 페이지 환경에 맞춰 `pathSegmentsToKeep = 1` 설정을 적용했습니다.
2. **메인 페이지 수정 (`index.html`)**:
   - `<head>` 섹션에 변환된 쿼리 스트링을 다시 원래의 URL 경로로 복구하는 스크립트를 추가했습니다.
3. **배포 가이드 수정**:
   - 해당 기능이 이미 적용되었음을 문서에 명시했습니다.

## 결과
- GitHub Pages 배포 후 어떤 경로에서든 새로고침을 하거나 직접 링크로 접속해도 정상적으로 앱이 로드됩니다.
