# 작업 내역 (2026-01-07)

## 요청 사항
- 목표 수정 모달에서 일일/주간/자유 유형일 때 기존 선택 날짜가 캘린더에 표시되지 않는 문제 해결.

## 수행 내용
1. **Calendar 컴포넌트 개선**:
   - `useEffect`를 추가하여 `selectedDate` 또는 `startDate` 프롭이 변경될 때 캘린더의 현재 표시 월(`currentMonth`)이 해당 날짜의 월로 업데이트되도록 수정했습니다.
2. **GoalEditorModal 로직 수정**:
   - 목표 수정 시 초기 데이터 로드 로직을 정교화하여 `selectedCalendarDate`가 정확히 설정되도록 했습니다.
   - `periodType` 변경 시 `selectedCalendarDate`가 오늘로 리셋되는 로직에 `goalToEdit` 및 `isOpen` 조건을 추가하여 수정 모드에서는 리셋되지 않도록 방지했습니다.

## 결과
- 목표 수정 시 이전 날짜가 캘린더에 정확히 표시되고 하이라이트됩니다.
