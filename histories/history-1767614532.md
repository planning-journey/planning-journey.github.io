## 2026년 1월 5일 (월) - 할 일 모달 UI/UX 개선

### 버그 내용 및 개선 사항

1.  **할 일 상세 모달에서 모달 높이가 뷰포트 높이를 넘지 않게 설정.**
2.  **할 일 상세 모달에서 설명 마크다운을 표시하기 위한 prose 클래스가 동작하지 않는 오류 수정.**
3.  **할 일 수정 모달에서 모달 높이가 뷰포트 높이를 넘지 않게 설정.**

### 해결 과정

1.  **모달 높이 제한 및 스크롤 기능 추가 (할 일 상세 모달 및 할 일 수정 모달):**
    *   `src/components/TaskDetailModal.tsx` 및 `src/components/TaskEditorModal.tsx` 파일의 메인 모달 컨테이너 `div`에 Tailwind CSS 클래스 `max-h-[90vh]`와 `overflow-y-auto`를 추가하여 모달의 최대 높이를 뷰포트의 90%로 제한하고, 내용이 넘칠 경우 스크롤이 가능하도록 설정.

2.  **마크다운 `prose` 클래스 미작동 오류 수정:**
    *   `tailwind.config.js` 파일에 `@tailwindcss/typography` 플러그인이 정상적으로 포함되어 있음을 확인.
    *   `src/index.css` 파일이 `@import "tailwindcss";` 대신 표준 Tailwind CSS 지시어 (`@tailwind base;`, `@tailwind components;`, `@tailwind utilities;`)를 사용하도록 수정. 이 변경을 통해 `@tailwindcss/typography` 플러그인에서 생성된 `prose` 관련 스타일이 제대로 적용되도록 함.

### 결론

할 일 상세 모달과 할 일 수정 모달의 높이 제한 문제를 해결하고, 할 일 상세 모달 내 마크다운 설명에 `prose` 클래스가 정상적으로 적용되지 않던 오류를 수정했습니다. 모든 변경사항 적용 후 `npm run build`를 성공적으로 완료했습니다.
