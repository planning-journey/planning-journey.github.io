1. **배포 가이드 생성 (`docs/github-pages-deployment.md`)**:

   - `public/CNAME` 파일 작성법.
   - GitHub Actions 워크플로우(`YAML`) 설정.
   - GitHub 저장소 설정 및 DNS CNAME 레코드 추가 방법 등을 포함했습니다.

## 결과
- 사용자가 직접 프로젝트를 서브 도메인에 배포할 수 있는 명확한 절차서가 마련되었습니다.
