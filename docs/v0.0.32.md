- OngoingGoals 하단은 영역은 화면의 나머지 영역을 채우도록 확장. 이 영역을 일일 상세 영역이라고 지칭.
  - `App.tsx`의 메인 컨테이너에 `flex flex-col`을, `main` 태그에 `flex-grow overflow-y-auto`를 적용하여 구현.
- 일일 상세 영역 하단에 Evaluation 이라는 이름의 제목을 가진 OngoingGoals와 유사한 영역 생성. 
  - `EvaluationHeader.tsx` 컴포넌트를 생성하여 구현.
  - chevron 버튼을 통해 토글할 수 있음.
    - chevron up/down 사용.
  - 토글 상태는 아직 미구현.
- Evaluation 위에는 하얀 배경으로 구분되는 또 하나의 폼 영역 구현.
  - `DailyDetailForm.tsx` 컴포넌트를 생성하여 구현.
  - 폼 영역에는 인풋창과 + 아이콘을 가진 버튼으로 구성.
- Evaluation과 폼 영역은 항상 하단에 고정.
  - `DailyDetailForm`과 `EvaluationHeader`를 포함하는 `div`에 `sticky bottom-0` 스타일을 적용하여 구현.
- 나머지는 일일 할 일 목록을 표시하기 위한 영역으로 남겨두기.
  - `App.tsx`의 `main` 태그 내 `div className="flex-grow p-4"` 영역이 일일 할 일 목록을 위한 영역으로 사용됨.
