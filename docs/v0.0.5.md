헤더의 Settings 버튼을 누르면 컨텍스트 메뉴 형태로 다크, 라이트, 시스템 옵션을 선택할 수 있게 해줘.
각 옵션 선택 시 그에 맞게 앱의 테마를 변경해야돼.
따라서 각 설정에 맞는 앱 테마 클래스도 설정해줘.

마지막으로, 앱 검증할 일이 있을 경우 바이트 서버가 구동되는 일은 없게 해줘.
네가 바이트 서버 구동 후 taskkill로 해당 작업을 종료하는 일을 종종 하는데,
바이트 버서가 구동되면 프로세스가 락에 걸려서 내가 직접 바이트 서버를 종료하게 되고,
그럼 너는 존재하지 않는 프로세스를 taskkill로 제거하려고 해서 무한 루프에 빠져.
