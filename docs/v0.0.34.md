- 일일 상세 영역 하단에 있는 폼에서 할 일 입력 후 폼 제출 혹은 + 버튼을 누르면 bottom sheet 오픈.
- bottom sheet은 할 일이 속한 목표를 설정할 수 있는 화면.
  - 오늘 날짜에 Active 상태인 목표들이 목록으로 표시되고, 목록 중에서 한 가지 선택.
  - 선택 안함도 선택 가능.
- 일일 상세 영역의 빈 공간은 할 일 목록 표시 공간.
  - 추가된 할 일은 여기에 표시.
  - 좌측에 체크박스, 할 일 이름이 위치하고 우측에 쓰리닷 표시.
  - 할 일 이름 위에 작게 목표 이름과 색상 표시.
  - 쓰리닷 클릭 시 수정/삭제 옵션이 있는 컨텍스트 메뉴 표시.
