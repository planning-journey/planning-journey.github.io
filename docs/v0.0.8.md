목표 에디터 모달에서 생성한 목표를 목록에 표시할 때 색상을 목표 좌측에 원형으로 표시.
목표 우측은 Pencil 아이콘과 X 아이콘을 배치해줘.
Pencil 아이콘은 클릭하면 목표 에디터를 열어서 목표를 수정할 수 있게 해줘.
X 아이콘은 클릭하면 confirm 모달을 열어서 삭제 여부를 확인하게 해줘.
마지막으로 목표 아래에는 목표의 기간 유형과 입력했던 날짜를 표시해줘.
- 연간 선택시 입력한 연도 표시
- 월간 선택시 입력한 연도와 월 표시
- 주간 선택시 입력한 주를 M월 N째 주 형태로 표시
- 일간 선택시 입력한 날짜를 표시
- 자유 선택시 입력한 시장일과 종료일 표시
이에 따라 데이터 구조는 변경하되, 모든 유형에서 시작일과 종료일 필드는 유지.
