목표 에디터 모달에서 목표 기간 선택에 따라 다음 기능 구현.

1. 연간
   - 연도 입력창 표시
   - 연간 목표는 입력된 연도의 1월 1일부터 12월 31일 까지가 목표 기간
2. 월간
   - 연도 입력창과 월 선택자 표시.
   - 월간 목표는 입력된 월의 1일부터 말일까지가 목표 기간
3. 주간
   - 달력 컴포넌트 표시.
   - 달력에서 날짜 선택시, 해당 날짜가 포함된 주 전체 선택.
   - 한 주는 월요일부터 일요일까지.
   - 주간 목표는 선택된 주의 월요일부터 일요일까지가 목표 기간.
4. 일간
   - 달력 컴포넌트 표시.
   - 달력에서 날짜 선택시 해당 날짜가 목표일.
   - 일간 목표는 선택된 날짜가 목표 시작일이자 종료일.
5. 자유
   - 달력 컴포넌트 표시.
   - 시작일과 종료일 범위 직접 선택.

모든 내용 입력 후 '목표 생성' 시 신규 목표 생성 후 IndexedDB에 반영.

생성된 목표는 즉시 앱 전체에 반영.
